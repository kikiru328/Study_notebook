1. 딥러닝에서 입력 데이터가 처리되는 상세 내용은 일련의 숫자로 이루어진 층의 가중치에 저장되어 있음.
   그 층의 가중치 = 파라미터 (parameter)
2. 신경망의 손실 함수 (loss function) 또는 목적 함수 (objective function)이 담당하는 것은 신경망의 출력을 제어하는 것.
3. 기본적인 딥러닝 방식은 피드백 신호를 사용하여 현재 샘플의 손실 점수가 감소되는 방향으로 가중치 값을 조금씩 수정하는 것. 이런 수정과정은
   딥러닝의 핵심 알고리즘인 역전파 알고리즘을 구현한 옵티마이저 (optimizer)가 담담함.
4. 네트워크가 모든 샘플을 처리하면서 가중치가 조금씩 올바른 방향으로 조정되고 손실 점수가 감소함 = 훈련 반복
5. 따라서 최소한의 손실을 내는 네트워크가 타깃에 가능한 가장 가까운 출력을 만드는 모델은 만들자 == 딥러닝
